# Ebbo's Portfolio - SvelteKit Edition

Bright OutRun arcade-inspired portfolio website for Bernhard Rode.

## Preview

Vibrant 80s arcade aesthetic with bright blue sky, neon sun, animated grid, and retro racing vibes inspired by classic OutRun games.

## Features

- Bright OutRun color palette (cyan, magenta, yellow, orange)
- Animated perspective grid floor
- Retro sun with horizontal bands
- Bright gradient sky
- Palm trees and racing car
- Neon glow hover effects
- Scroll-triggered fade animations
- Fully responsive
- Built with SvelteKit

## Run Locally

```bash
cd sveltekit-portfolio
npm install
npm run dev
```

Open http://localhost:5173

## Build for Production

```bash
npm run build
npm run preview
```
